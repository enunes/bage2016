Aprendendo a aprender Software Livre

Sempre teve curiosidade de conhecer o Debian, mas só sabe baixar a ISO do Ubuntu?
Sempre quis reportar um bug, mas não sabe nem o que é o Bugzilla?
Sempre teve interesse de contribuir para algum projeto, mas não sabe como contatar os responsáveis?
Sempre abre a linha de comando, mas só sabe usar o apt-get?

Se você respondeu sim para alguma das perguntas acima, essa palestra é para você!

Aprendendo a aprender Software Livre é uma palestra cujo foco é desmistificar os "primeiros passos" daqueles que tem interesse em usar Software Livre. A ideia é abordar os melhores locais para obter ajuda, como receber informações precisas e atualizadas, aprender a se comunicar com as mais diversas comunidades e por fim também realizar uma contribuição para algum projeto de código aberto.

O foco desta atividade não é o uso do Software Livre em si, mas sim como descobrir a maneira mais efetiva de procurar soluções e agregar conhecimentos e respostas a comunidade.

Por que implantar o suporte de Helpdesk com a ferramenta GLPI?

GLPI (Gestionnaire Libre de Parc Informatique) é uma ferramenta de gestão de parque de informática, mundialmente utilizada, baseado na biblioteca de boas práticas ITIL, que provê diversos recursos dentre eles o serviço de assistência de chamados. Neste espaço será apresentado os processos que motivam a sua implantação, através da sua definição, instalação, configuração e validação especificada através de um estudo. Deste, adota-se a funcionalidade de um setor de HelpDesk com fluxo definido para atendimento (assistência de chamados TIC) da UNIPAMPA. Ao fim, são apresentados os resultados obtidos do processo e, possibilidades futuras, através das contribuições que a ferramenta entrega para a melhoria dos serviços de TIC prestados.

Buildroot - Criação de um sistema Linux Embarcado para a BeagleBone

O Buildroot (www.buildroot.org) é uma ferramenta simples, eficiente e fácil de se utilizar, capaz de construir sistemas Linux completos através de cross-compilação. O Buildroot é capaz de gerar todos os componentes necessários para a construção do sistema em alguns minutos, incluindo o conjunto necessário de ferramentas para compilação, um kernel Linux e as aplicações desejadas, em um processo customizável através de centenas de pacotes de software livre disponíveis.

Esta palestra apresenta uma introdução ao Buildroot e mostra como utilizá-lo para criar um sistema Linux customizado, através de uma demonstração prática de geração de um sistema Linux Embarcado para uma placa de desenvolvimento BeagleBone (www.beagleboard.org/bone).

Como posso colaborar com o PostgreSQL?

O PostgreSQL é um projeto de software livre mantido por uma comunidade ampla, formada por pessoas de vários lugares do mundo. A intenção dessa apresentação é exemplificar as diversas formas possíves de colaboração e como você pode iniciar hoje mesmo a fazer parte desta comunidade e fazer com que este projeto nunca acabe.

Não só de desenvolvedores (hackers) se faz uma comunidade de software livre, você pode se engajar:
  • curte escrever? então ajude na documentação e/ou escreva sobre o PostgreSQL em seu blog;
  • tem experiência com tradução? ajude o time de tradutores com o processo de tradução (documentação oficial em pt-br é uma deficiência nossa);
  • curte testes e revisão de código? então, precisamos de você para melhorar o processo de liberação de releases com a melhor qualidade possível;
  • é um empresário/investidor e se beneficia desse fantástico produto? invista em eventos (sempre precisamos de patrocinadores) e até mesmo em outros profissionais que desejam colaborar com o PostgreSQL mas precisam usar suas horas vagas para esse projeto;
  • é um usuário PostgreSQL experiente? então ajude outros a ficar experiente como você.

    • Engajamento é a palavra-chave para o sucesso do PostgreSQL e de qualquer projeto. Envolva-se, engaje-se, o PostgreSQL precisa de você!
